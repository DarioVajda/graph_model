# Research Project: Toward Native Unifying of Graph and Text

This repository contains the research proposal for a novel model architecture designed to bridge the gap between traditional Large Language Models (LLMs) and Graph Transformers (GTs). The goal is to create a model capable of operating natively on both sequential text and graph-structured data.

## 1. Motivation
Standard LLMs are limited by their reliance on sequential data. This research aims to unify text and graph modalities to unlock new capabilities, such as:
* **Enhanced mathematical reasoning** 
* **Protein analysis and molecule generation** 
* **Native processing of complex relational data structures** 

## 2. Proposed Architecture
The model represents all data as a global graph structure where nodes and edges contain text tokens. Key innovations include:

### Graph-aware Positional Encodings (PE)
The project introduces a combination of **Rotary Positional Embeddings (RoPE)** and graph-specific positional encodings. By utilizing the eigenvectors of the graph Laplacian, the model constructs joint coordinates that account for both the relative position of a token within a node and the global structural displacement in the graph.

### Inference Algorithm
1.  **Node Generation**: Generate node text autoregressively.
2.  **Edge Generation**: Use similarity between node summary tokens and a virtual `STOP` token to determine edge endpoints.
3.  **Attribute Generation**: Generate textual properties for each edge.

## 3. Training and Evaluation
The model is fine-tuned from a pretrained LLM (1B to 3B parameters) using **continual pretraining** on graph-structured data.

### Optimization Objective
The training objective maximizes the negative log-likelihood for next-token prediction (node/edge text) and maximizes similarity between node summaries for valid edge pairs.

### Evaluation Benchmarks
* **Textual**: ARC-e/c, OpenBookQA, MMLU, GSM8K, and BBH.
* **Graph**: OGBN-Arxiv (node classification) and MoleculeNet (molecule classification).
* **Comparative**: Benchmarked against Topological Pre-training of Language Models.
